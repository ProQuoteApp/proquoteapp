# User Experience Guidelines

## Design Principles
### Accessibility
- Color contrast
- Text sizing
- Screen reader support
- Navigation assistance

### Usability
- Intuitive navigation
- Clear hierarchy
- Consistent patterns
- Error prevention

6. Payment Receipt

### Mobile-First
- Touch targets

- Responsive layouts
